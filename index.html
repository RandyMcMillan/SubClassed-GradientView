Subclassed UIView - GradientView

Create

~/Library/Developer/Xcode/Templates/File Templates/

if it doesn't exist

Drop into: 

~/Library/Developer/Xcode/Templates/File Templates/

SubClass UIView and add Gradient styling. You can also add other properties and this is a great start custom UI styling for iOS
